# Astrolo
.Net components for GeneKeys and Human Design applications, including WPF controls for visualizing:
- HD Mandala (including Astrology signs)
- HD Bodygraph (including Centers, Gates, and Channels)
- GeneKeys Profile
- YiJing Hexagram
